#POPUED team web

------

*项目下载*
>$git clone https://github.com/popued/popweb.git

*项目本地运行*
>cd popweb  
$hexo generate  
$hexo server  

*项目提交*  
>$hexo clean  
$git add "修改或新建文件路径"  
$git commit -m "文件描述"  
$git push origin master

*项目更新*  
>$git pull origin master 

*项目部署到github.io*
>$hexo deploys
